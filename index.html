Problema Seleccionado:

Introducción

Actualmente, aprender inglés es muy importante tanto para estudiar como para el futuro laboral. Aun así, muchas personas tienen dificultades para aprender vocabulario porque los métodos tradicionales suelen ser aburridos y poco interactivos. Por esta razón, se desarrolló un juego educativo que ayuda a aprender inglés de una forma más dinámica y entretenida. Para crear la solución, se aplicaron los siete pasos del método del ingeniero, organizando el proceso desde el análisis del problema hasta el desarrollo final. Como resultado, se creó un juego tipo “ahorcado” en inglés, donde los usuarios pueden aprender nuevas palabras mientras juegan e interactúan con el sistema. En este sitio web se muestra todo el proceso realizado, desde la identificación del problema hasta la implementación y funcionamiento final del juego.
